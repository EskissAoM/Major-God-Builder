# AoM:R Major God Builder

**AoM:R Major God Builder** is a static browser tool for creating custom major gods for **Age of Mythology: Retold**.

It lets users choose a pantheon, god power, unique technology, bonuses, minor gods, and culture-specific options, then exports a ready-to-install local mod ZIP. Presets can also be exported and loaded as JSON files, making custom gods easy to share.

> This is a fan-made modding tool and is not affiliated with Microsoft, World's Edge, Forgotten Empires, or any Age of Mythology rights holder.

## Features

- Create a custom major god from existing Age of Mythology: Retold content.
- Choose pantheon, name, title, focus, icon, god power, unique tech, bonuses, and minor gods.
- Supports Greek, Egyptian, Norse, Atlantean, Chinese, Japanese, and Aztec cultures.
- Handles culture-specific choices such as Greek heroes, Chinese Mythic heroes, and Aztec forms.
- Generates GodPicker and TechTree files dynamically from the selected configuration.
- Exports a complete local mod ZIP directly from the browser.
- Exports and loads shareable preset JSON files.
- Includes a visual god preview that can be exported as an image.

## Installing a generated mod

1. Generate and download the mod ZIP.
2. Unzip the folder.
3. Move the unzipped mod folder to:

   ```text
   C:\Users\UserName\Games\Age of Mythology Retold\SteamID\mods\local
   ```

4. Start Age of Mythology: Retold and enjoy.
