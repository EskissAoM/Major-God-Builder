AoM Retold Major God Creator

This draft lets you choose a pantheon, any existing Archaic Age god power, and two existing minor gods per age, then generates a mod ZIP for a custom major god.

Draft status: this outputs major_gods_mods.xml, techtree_mods.xml, stringmods.txt, and GodPicker/TechTree XAML files. The pregame GodPicker and TechTree files use compact generated Archaic Age power blocks and real vanilla blocks for the selected minor gods. It is still a test generator: major-god bonuses can now be selected from the implementation map, but some complex mixed-file bonuses may still need in-game iteration.

1. Major god

Internal name/folder is generated automatically from the display name.

Greek-only choices

Choose one hero for each age from that age's Greek hero pool, plus one Fortress unique unit.

Chinese-only choices

Choose one Mythic Age special hero.

Aztec-only choices

Choose one Classical Teixiptla Form and one Mythic Incarnate choice.

2. Unique technologies

Choose up to two major-god unique technologies. The list is filtered by the selected pantheon, with special availability rules where needed.

3. God bonuses

Choose up to four major-god bonuses. The list is filtered by the selected pantheon using the compatibility column from the spreadsheet.

4. Minor gods

Choose exactly two existing minor gods for each age. By default the list is filtered to the selected pantheon, but you can unlock all pantheons for testing.
